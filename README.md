# Ledger

1. Navigate to the location of the project and run "node ledger.js". Thats the main file.
2. The server will start.
3. In the browser, navigate to http://localhost:3000/
4. A form comes up to request dates and rent of the tenant.
When the details are filled and submitted, it navigates to a page which displays ledger in the form of a table.
5. End.
